# **App Name**: Ink & Asphalt

## Core Features:

- Profile Creation: Profile creation with sections for name, favorite book, favorite book quote, preferred match, age, country, gender, and a selection of hobbies. Users can upload pictures and videos of themselves.
- Profile Search: Search and filter profiles by age, country, and gender.
- Direct Messaging: Private messaging system to connect with matches.
- Matching System: A scrolling interface to like or pass on potential matches, coupled with mutual match notifications.

## Style Guidelines:

- Primary color: Deep red (#800020), inspired by the intense passion found in classic literature and the rebellious spirit of bikers.
- Background color: Soft beige (#F5F5DC), offering a neutral backdrop reminiscent of aged book pages.
- Accent color: Burgundy (#4B0000) for highlights, referencing the glow of a campfire under a starlit sky or a setting sun on the open road.
- Secondary Accent Color: Crimson (#9B1B30)
- Classic, slightly condensed serif fonts for profile text, balancing the literary theme with readability.
- Use simplified line icons that represent books, motorcycles, and related activities.
- Split-screen layouts for profile viewing, juxtaposing interests and photos.
- Subtle transitions and loading animations.
